### 🐟기능소개 (1차 MVP)
![fish tank](docs/asset/fish_tank.png)

#### 📖물고기 도감  
- 잡은 물고기 판별
- 수집한 물고기 정보 저장 (어종 및 사진, 크기 등)
- 낚시 중 물고기 등록이 편리한 위젯
- 수집한 물고기를 확인하고, 자랑할 수 있는 가상 어항
- 물고기 등급표
- 물고기 크기 판별
#### 🚩낚시 포인트 정보 제공 (지도 형식)
- 포인트 별 물때, 날씨, 수온, 출몰 어종 등 낚시에 필요한 정보 제공
- 낚시 포인트 공유 기능
